Install the program by running the executable.

Copy and paste /plugins and config.ini into the new directory, which should be
    \Program Files\MyceliumHub.
    The directory should be like this:
    MyceliumHub/
        app
        plugins
        runtime
        MyceliumHub.exe
        MyceliumHub.ico
        config.ini
